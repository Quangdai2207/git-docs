# GIT COMMANDS

- ## Git Config

| STT   | Commands                                    | Chuc nang                                                                                   |
| ----- | ------------------------------------------- | ------------------------------------------------------------------------------------------- |
|       | **Kiem tra Configs**                        |                                                                                             |
| **1** | git config --list                           | Liet ke toan bo thong tin cau hinh khong bao gom file path                                  |
| **2** | git config --list --show-origin             | Liet ke toan bo thong tin cau hinh bao gom file path                                        |
| **3** | git config --show-origin user.name          | Lay thong tin cau hinh cu the bang thuoc tinh va xac dinh cap do cau hinh dang duoc ap dung |
| **4** | git config user.name                        | Lay thong tin cau hinh bang thuoc tinh                                                      |
|       | **Thiet lap Config**                        |                                                                                             |
| **5** | git config --global user.name username      | Thiet lap thong tin cau hinh cho thuoc tinh user.name                                       |
| **1** | git config --global user.email email        | Thiet lap thong tin cau hinh cho thuoc tinh email                                           |
| **1** | git config --global init.defaultBranch main | Thiet lap thong tin ten nhanh mac dinh cho GIT                                              |
| **1** | git config --global core.editor nano        | Thiet lap trinh Editor cho Git                                                              |
